## エッジAIの電力異常

施設の電力メーターデータをリアルタイムで監視し、
**設備故障**と**センサー誤計測**を自動で識別するAIシステムです。

### 背景・課題

工場やオフィスビルでは電力の異常上昇が機器故障や火災リスクにつながりますが、
「本当に設備が壊れているのか」「センサーが誤作動しているだけなのか」を
人手で判断するのは困難でした。本システムはその判別を自動化します。

### 技術的なポイント

**2モデル組み合わせによる3クラス分類**

| AEスコア | LSTMスコア | 判定 |
|----------|------------|------|
| 低       | 低         | 正常 |
| 高       | 高         | 設備故障（機器の動作パターン自体が異常）|
| 高       | 低         | センサー誤計測（計測値は歪むが時系列は正常）|

- **オートエンコーダー**：60分の電力波形を圧縮・復元し、復元誤差を異常スコアとして利用
- **LSTM**：過去60分から次の値を予測し、予測誤差をスコアとして利用
- **One-class学習**：正常データのみで訓練することで、未知の異常にも対応
- **故障データ合成**：冷蔵庫・扇風機・PC等5種の実機故障パターンを文献ベースで設計

### 技術スタック

Python / PyTorch / pandas / NumPy / LEAD1.0 Dataset
